RAW B2C LTD WEBSITE PROJECT SUMMARY
Goal:
Build a professional company profile website for RAW B2C LTD with an admin system.
Technology Stack:
Frontend:
- HTML5
- Tailwind CSS
- JavaScript
- Responsive design
- Animations
Backend:
- PHP
- MySQL
Hosting:
- Shared hosting (Hostinger recommended)
- cPanel
- MySQL database
Design:
- Green and white corporate theme
- Premium modern UI
- Smooth animations
- Hover effects
- Scroll reveal animations

4. Blog
- News
